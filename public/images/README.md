# Images Folder

This folder contains static images for the Beagle Property Portal.

## Usage

Images placed here can be accessed at `/images/filename.ext`

Example:
```tsx
<img src="/images/logo.png" alt="Logo" />
```

## Recommended Structure

```
public/
  images/
    logos/          - Beagle logos and brand assets
    icons/          - UI icons
    properties/     - Property photos
    avatars/        - User avatars
    placeholders/   - Placeholder images
```

## File Naming Convention

- Use lowercase
- Use hyphens for spaces: `beagle-logo.png`
- Include dimensions for multiple sizes: `logo-sm.png`, `logo-lg.png`
